About This Project

This is a simple demo banking project created for learning purposes and to showcase basic banking functionalities such as account creation, deposit, withdrawal, and balance overview.

The project is built using Python for backend, SQLite for data storage, and HTML/CSS for frontend. The design is inspired by modern banking websites.

All features are simplified and serve as a practice project for demonstrating a banking system workflow.
